
H1-1

–
(Palindromes) A palindrome is a sequence of characters that reads the same backward as forward.
–
For example, each of the following seven-digit integers is a palindrome: 1123211, 3555553, 2455542 and 8116118. Write an application that reads in a seven-digit integer and determines whether it is a palindrome. If the number is not seven digits long, display an error message and allow the user to enter a new value.


H1-2

–
Write an application that reads in an positive integer that is less than Integer.MAX_VALUE and determines whether it is a palindrome. (數字無位數限制)
–
If the length of the number is not an odd number or the number is out-of-range, display an error message and allow the user to enter a new value.
